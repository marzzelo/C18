IMPORTANTE!!! CREACION DE LIBRERIAS

- Cargar el workspace obj.mcw
- Agregar el proyecto conteniendo todos los módulos fuentes .c y el header correspondiente.
- Seleccionar el dispositivo en particular para el que se crea la librería.
- Establecer el path de salida .lib como C:\@c18\lib\P18<disp>\
- Establecer el tipo de compilación a "Build Library Target" SIN MARCAR la opcion "Generic Library" ni "Extended"
- Agregar el proyecto a VSS
- Verificar compilación y que existe la salida en C:\@c18\lib\P18<disp>\<nombre>.lib
